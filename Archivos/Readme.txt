Modo de Ejecución
*****************
1. Comandos de Ejecucón.
---Comando que se requiere para ejecutar el test automatizado:
** mvn clean
** mvn install

Al ejecutar estos dos comando se limpia y se genera un nuevo reporte
-- Como visualizar reporte.
** Ingresamos a la carpeta targer.
** ubicamos la rama site/serenety
** buscamos index.html luego presionas clic derecho y seleccionas open in/browser y eliges el navegador.
** por ultimo presinamos en la pestaña Test Results para validar las evidencias generadas.
